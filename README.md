# Wrangle

A lightweight native macOS daily/weekly planner built with SwiftUI.

## Features

- Day and week views for planning
- Calendar.app two-way sync
- Notification reminders
- Menu bar quick access
- Local SwiftData persistence

## Requirements

- macOS 14.0+ (Sonoma)
- Xcode 15+

## Getting Started

1. Open `Wrangle/` folder in Xcode
2. Select your team in Signing & Capabilities (or disable signing for local use)
3. Build and run (Cmd+R)

On first launch, the app will request calendar and notification permissions.
